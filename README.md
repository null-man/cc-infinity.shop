# CC Infinity Shop

现代化电商网站模板，基于 Next.js 构建。

## 技术栈

- Next.js 14
- React 18
- Tailwind CSS
- TypeScript

## 一键部署到 Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/null-man/cc-infinity.shop)

## 本地开发

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev
```
